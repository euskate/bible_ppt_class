# 실행
- 바탕화면에 있는 "오전예배PPT만들기" 바로가기 파일을 실행합니다.
- 뒤에 콘솔창(검은화면)이 뜨는 것은 신경쓰지 마세요. (개발중입니다.)
   - 신경쓰이면 최소화로 감춰주세요. 콘솔창을 끄면 프로그램이 꺼집니다.

# 버전 변경사항 (v0.03)
- 전체 실행 버튼(원터치)을 추가하였습니다.
- 전체 실행 시 날짜 변경 추가(실행 당일 날짜로 변경)
- 교독문 오류 수정(복사 > 입력으로 변경)
- 전체 실행 시 자동으로 다른이름으로 저장합니다.

# 수작업으로 해야하는 것
- 말씀 선포 본문에 색깔을 입히는 것 (설교 한글 파일을 참고하세요.).
- 사진과 동영상자료를 추가하는 것 (카카오톡 예배팀방을 참고하세요).
- 마지막에 광고를 입력 (주보를 참고하세요.).
- 작업 후 전체 검토 필요.
- (X:전체실행시 자동변경) 맨 첫번째 슬라이드에 날짜를 변경 (예배 당일 날짜).
- (X:전체실행시 자동저장) 작업 후 다른이름으로 저장. (경로는 바탕화면\오전예배.pptx를 권장합니다.)

# 주의해야 할 점
- 제대로 실행이 되지 않을 경우, 열려있는 파워포인트 창을 모두 닫고 새로 실행해보세요.

# 기능 설명
0. 전체 실행(원터치)
- 모든 항목을 입력한 후 실행합니다.
- (전체실행 시 소스 ppt 열기도 실행하지 마세요.)
- 입력되지 않았거나 잘못입력된 것은 무시하고 실행됩니다.
- 전체실행 시 날짜가 자동변경됩니다.
- 실행 마지막에 바탕화면에 자동저장됩니다.

1. 소스 ppt 열기
- 편집할 견본 파워포인트 파일을 엽니다.
- 소스파일은 수정하지 말고, 새 이름으로 저장하기 해주세요.

2. 교독문 입력
3. 첫번째 찬송가 입력
5. 두번째 찬송가 입력

- 그날 주보에 있는 번호를 입력하고 버튼을 누릅니다.
- 원본 경로에서 파워포인트를 복사해서 소스 ppt에 붙여넣습니다.
- 숫자만 입력하세요. (1~645)

4. 본문입력
- 목사님이 주시는 한글 파일을 복사해서 넣고, 버튼을 누릅니다.
- 성경봉독과 본문을 자동으로 입력해 줍니다.

6. 전환애니메이션 추가
- 모든 작업이 끝난 후 버튼을 누르면 모든 슬라이드에 부드러운 전환효과가 추가됩니다.
- (빠뜨려도 상관없습니다.)
